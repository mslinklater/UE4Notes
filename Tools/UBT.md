# Unreal Build Tool (UBT)

Handles the build process.

## Notes

This is usually run from the buid batch files found in Engine/Build/BatchFiles/

## Command line Parameters

**Arguments[0]**

The target name

**Arguments[1]**

Platform (eg Win64)

**Arguments[2]**

Configuration (eg Development)

**Arguments[3]**
